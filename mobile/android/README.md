# MDS Android Companion

Source-only native Android shell for explicit screenshot selection and microphone voice notes. New node streams are quarantined by the Command Centre pairing gate. The app has no camera, background service, screen-capture, media-projection, system-audio, provider, execution, or external-send authority.

Native build status is `BLOCKED_TOOLCHAIN_UNAVAILABLE` on the current machine because Java, Gradle, Android SDK, ADB, emulator, and signing evidence are unavailable. The default `10.0.2.2` gateway URL is emulator-local development only. Real-device transport requires a separately approved authenticated TLS/LAN design; do not expose the loopback server directly.
